### NextJS steps to launch dev environment

1. npm install
2. npm run dev

### NextJS steps to launch production environment

1. npm install
2. npm run build
3. npm run start

### NextJS steps to launch in docker

1. docker build -t technius-production .
2. docker run -p 80:80 technius-production
